# 용어 정의

## Course

사용자가 생성한 러닝 코스입니다.

코스는 지도 위의 경로 좌표 목록, 제목, 설명, 거리, 예상 소요 시간, 공개 여부 등을 포함합니다.

## Route Point

코스를 구성하는 단일 좌표 지점입니다.

모든 Route Point는 WGS84 기준의 위도, 경도 값을 사용합니다.

## Path

Route Point의 순서 있는 배열입니다.

사용자가 그린 경로의 실제 형태를 표현합니다.

## Bounds

지도에서 현재 보고 있는 사각형 영역입니다.

코스 탐색 API는 Bounds를 기준으로 해당 영역과 겹치는 코스를 조회합니다.

## GPX

GPS Exchange Format입니다.

Runvas는 저장된 코스를 외부 러닝 앱에서 활용할 수 있도록 GPX 파일로 내보냅니다.

## Estimated Duration

코스를 완주하는 데 걸릴 것으로 예상되는 시간입니다.

MVP에서는 경로 탐색 API 또는 모바일 표시 로직이 만든 예상 시간을 서버가 저장하고 응답합니다.

## Visibility

코스 공개 범위입니다.

MVP에서는 `PUBLIC`, `PRIVATE` 두 값을 사용합니다.

## User

Runvas에 로그인하는 사용자입니다.
MVP에서는 카카오 소셜 로그인으로 생성됩니다.

코스, 게시글, 댓글, 좋아요의 주체입니다.

## Public Profile

다른 사용자에게 노출되는 사용자 정보입니다.

이메일 같은 로그인 정보는 포함하지 않고, 닉네임, 프로필 이미지, 소개만 포함합니다.

## Post

사용자가 커뮤니티에 작성하는 게시글입니다.

게시글은 텍스트를 중심으로 하며, 선택적으로 공개 코스를 첨부할 수 있습니다.

## Comment

게시글에 달리는 댓글입니다.

MVP에서는 게시글 댓글만 지원하고, 대댓글은 지원하지 않습니다.

## Like

사용자가 공개 코스 또는 게시글에 남기는 긍정 반응입니다.

동일 사용자는 같은 대상에 좋아요를 한 번만 누를 수 있습니다.
