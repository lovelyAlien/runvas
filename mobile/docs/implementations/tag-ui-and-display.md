# 태그 검색 UI 개선 및 태그 표시 기능

## 개요

- 코스 검색 UI 개선: 토글 버튼 제거, `#` 접두사 자동 태그 모드 전환 (A+C 조합)
- 코스 상세 화면에 태그 칩 표시
- 내 코스 / 북마크 목록에 태그 칩 표시
- 재사용 `TagList` 컴포넌트 신규 생성

## UI 변경사항

### CourseSearchBar
- placeholder: `"코스 이름 또는 #태그로 검색"`
- `#`으로 시작하면 자동으로 태그 검색 모드 (pricetag 아이콘으로 전환)
- 태그 모드 maxLength: 21 (`#` + 최대 20자)
- 이름 모드 maxLength: 100
- 검색 결과 아이템에 태그 칩 표시

## 테스트 시나리오

### 1. 태그 검색 자동 감지 (검색바)

| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | 지도 → 돋보기 버튼 탭 | 검색바 열림, placeholder "코스 이름 또는 #태그로 검색" |
| 2 | "한강" 입력 | 이름 검색 모드 유지 (search 아이콘), 코스 제목 부분 일치 결과 표시 |
| 3 | 입력 지우고 "#한강" 입력 | pricetag 아이콘으로 전환, "한강" 태그 정확 일치 코스 검색 |
| 4 | "#" 만 입력 | 검색 미실행, 결과 없음 표시하지 않음 |
| 5 | "#a" 21자 초과 입력 시도 | maxLength 21자에서 입력 차단 |
| 6 | 결과 없을 때 | `"#한강" 태그의 코스가 없습니다` 메시지 표시 |
| 7 | 검색 결과 아이템 | tags 있으면 칩 표시, 없으면 공간 없음 |

### 2. 코스 상세 화면 태그 표시

| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | 태그 있는 코스 상세 진입 | likeBar 아래에 `#태그명` 칩 표시 |
| 2 | 태그 없는 코스 상세 진입 | 태그 영역 미표시 (TagList 렌더 안 함) |
| 3 | 태그 여러 개 | 가로 wrap 형태로 여러 칩 나열 |

### 3. 코스 목록(내 코스/북마크) 태그 표시

| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | 내 코스 탭 진입 | 태그 있는 코스 아이템에 칩 표시 |
| 2 | 북마크 탭 진입 | 동일하게 태그 칩 표시 |
| 3 | 태그 없는 코스 | 칩 영역 미표시 |
