# Runvas Docs

`runvas-docs`는 `runvas-backend`와 `runvas-mobile`이 공통으로 참조하는 기술 스펙 문서 모음입니다.

이 저장소는 제품 소개나 회의록이 아니라, 두 프로젝트가 반드시 같은 기준으로 구현해야 하는 계약을 관리합니다.

## 프로젝트 개요

Runvas는 사용자가 지도 위에서 직접 러닝 코스를 설계하고 공유할 수 있는 위치 기반 러닝 코스 플랫폼입니다.

기존 러닝 서비스가 거리, 속도, 칼로리 같은 운동 기록 관리에 집중한다면, Runvas는 사용자가 하트, 별, 동물, 캐릭터 같은 형태의 코스를 만들고 공유하며 러닝 자체를 하나의 창의적인 경험으로 즐길 수 있게 하는 데 집중합니다.

## 저장소 역할

이 저장소에는 다음처럼 백엔드와 모바일이 함께 맞춰야 하는 내용만 둡니다.

- API 계약
- 데이터 모델
- 좌표계 및 거리 계산 기준
- 지도 범위 조회 규칙
- GPX 다운로드 규칙
- 커뮤니티 게시글, 댓글, 좋아요, 신고 규칙
- 에러 응답 형식
- 용어 정의

다음 내용은 각 구현 저장소에서 관리합니다.

- 화면 상세 UI 구현
- 서버 내부 아키텍처
- DB 마이그레이션 파일
- 배포 환경별 설정값
- 플랫폼별 라이브러리 사용법

## 변경 원칙

공통 계약이 바뀌면 `runvas-backend`와 `runvas-mobile` 모두 영향을 받습니다.

따라서 아래 변경은 반드시 이 저장소 문서를 먼저 수정한 뒤 구현합니다.

- API 요청/응답 필드 추가, 삭제, 이름 변경
- 데이터 타입 변경
- 좌표 순서, 단위, 거리 계산 방식 변경
- 코스 공개 범위나 조회 조건 변경
- GPX 파일 구조 변경
